Задачей агента определения производных для указанного цвета является определение производных цветов для указанного цвета. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу определение производных заданного цвета. Аргументом запроса является цвет, для котрого необходимо найти производные цвета. Результатом выполнения команды является графовая конструкция.
